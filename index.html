04 / INSTALLATION

安裝與執行

⚠️ 需要足夠 RAM：LLaMA3 約需 8GB，Gemma3:27b 約需 20GB。建議在 Mac M 系列或有 GPU 的機器上執行。
# 1. 安裝依賴 pip install flask chromadb sentence-transformers ollama # 2. 安裝 Ollama 並拉取模型 curl -fsSL https://ollama.com/install.sh | sh ollama pull llama3 ollama pull qwen3:8b ollama pull gemma3:27b # 3. 抓取資料並建立向量資料庫 python fetch_all_fda.py python fetch_pubmed.py python fetch_rxnorm.py python build_db_final.py # 4. 啟動 python app.py # → http://localhost:8000
